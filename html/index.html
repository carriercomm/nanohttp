nanoHttp :: a small perl-webserver

This is the home of the nanoHttp web server, a tiny but powerfull http server written in perl.

  • easy to install, configure and understand
  • small footprint
  • can run under a different user/group (default is nobody/nobody)
  • supports dynamic web pages with perl
